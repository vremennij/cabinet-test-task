# Тестовое задание Backend

# Описание
Ваша задача реализовать микросервис, который позволит неавторизованным пользователям подавать заявки на реализацию проектов,
а администраторам принимать или отклонять полученные заявки.

API-контранкт сервиса приведён в файле `api.json`.

# Технические требования
1. Сервис должен быть написан на GO
2. При реализации должен быть использован стандартный HTTP-роутер из `net/http`. 
3. Можно использовать любые дополнительные библиотеки
4. Сервис должен работать на порту `8000`

# Формат сдачи
1. Прикрепите ссылку на публичный репозиторий на GitHub или GitLab
2. В репозитории должен содержаться файл `compose.yaml`, который поднимет все контейнеры, необходимые для работы приложения (как минимум сам сервис и базу).
